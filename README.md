# 🎨 Obys Agency Clone

### A Pixel-Perfect Frontend Recreation of the Award-Winning Obys Agency Website

<p>
Built with <strong>HTML</strong>, <strong>CSS</strong>, <strong>JavaScript</strong>, <strong>GSAP</strong>, <strong>Locomotive Scroll</strong> and <strong>Shery.js</strong>
</p>

<br>

🌐 **Live Demo:** https://obys-agency-ruddy.vercel.app/

</div>

---

# ✨ Features

- 🎬 Animated Loading Screen
- 🖱️ Custom Cursor
- 🧲 Magnetic Navigation
- 🚩 Interactive Flag Animation
- 🌊 Smooth Scrolling using Locomotive Scroll
- ⚡ GSAP Timeline Animations
- 🖼️ Gooey Image Hover Effects (Shery.js)
- 🎥 Interactive Video Section
- ✍️ Typography Inspired by Obys Agency
- 📱 Responsive Layout
- 🎨 Modern UI & Micro Interactions

---

# 🛠 Tech Stack

| Technology | Purpose |

| HTML5 | Structure |

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yashbsl/Obys-Agency.git
```

Go into the project

```bash
cd Obys-Agency
```

Open

```
index.html
```

or run with Live Server.
